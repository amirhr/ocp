# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, as its Code of Conduct.

The full text is published at:
**https://www.contributor-covenant.org/version/2/1/code_of_conduct/**

That document, by reference, governs all participation in this repository, including Issues, Discussions, Pull Requests, code review, and any other project space.

## Scope

This Code of Conduct applies within all project spaces, and also when an individual is officially representing the project in public spaces.

## Reporting a concern

To report unacceptable behavior privately, open a **private security advisory** in this repository:

**Security tab → "Report a vulnerability"**

Despite the "security" framing of the GitHub UI, this channel is the preferred way to reach the maintainer about anything that warrants a private conversation, including Code of Conduct concerns. Reports submitted this way are visible only to the reporter and the project maintainer.

Until a neutral governance body is established, reports are received and acted on by the project maintainer. Once governance moves to a neutral home (see [GOVERNANCE.md](./GOVERNANCE.md)), enforcement responsibility transfers to that body.

All reports will be reviewed and investigated promptly and fairly. The reporter's identity will be kept confidential to the extent consistent with effective enforcement.

## Why a pointer rather than a full copy

Embedding the full Contributor Covenant text in this repository creates a maintenance burden (keeping it in sync with upstream revisions) without adding clarity. The canonical URL is stable, versioned, and authoritative. Adopting v2.1 by reference means contributors get the current authoritative text from the people who wrote and maintain it.
